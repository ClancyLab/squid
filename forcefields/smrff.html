smrff¶

squid.forcefields.smrff.parse_pfile(fname)[source]¶

This function will, given a smrff parameter file, will parse it by removing comments, trailing whitespaces, and empty lines.

Parameters

fname: str
The name of the parameter file to be parsed.

Returns

parsed: str
A parsed string of said parameter file.
