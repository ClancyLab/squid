class squid.structures.molecule.Molecule(atoms, bonds=[], angles=[], dihedrals=[], molecule_index=None, assign_indices=True)[source]¶

A molecule object to store atoms and any/all associated interatomic connections.

Parameters

atoms: list, structures.atom.Atom
A list of atoms.
bonds: list, structures.topology.Connector , optional
A list of all bonds within the system.
angles: list, structures.topology.Connector , optional
A list of all angles within the system.
dihedrals: list, structures.topology.Connector , optional
A list of all dihedrals within the system.
molecule_index: int, optional
The index to be assigned for this molecule.
assign_indices: bool, optional
Whether to assign the molecule a default index of 1 (if no other is specified), and to assign the atomic indices, indexed at 1. If molecule_index is specified, that will take precedence over the default of 1; however, the atom re-indexing will still take place.

Returns

molecule: structures.molecule.Molecule
The Molecule class container.

merge(other, deepcopy=False)[source]¶

This function merges another molecule into this one, offsetting indices as needed. When merging, the atom indices of this molecule is reassigned. Futher, if deepcopy is False, then the atom indices of the other molecule are also reassigned.

Parameters

deepcopy: bool, optional
Whether to merge via a deep copy, in which atoms are replicated (that is, the atom pointers are different between the molecules), or to merge via pointers, in which the other molecule has similar pointers.

Returns
