Reads an opls parameter file written in the Tinker file format.

Parameters

parameter_file: str, optional
Relative or absolute path to an opls parameter file, written in the Tinker file format.
pair_style: str, optional
The pair style to be assigned.

Returns

atom_types: list, dict
A list of the forcefield types for atoms, stored as a dictionary.
bond_types: list, dict
A list of the forcefield types for bonds, stored as a dictionary.
angle_types: list, dict
A list of the forcefield types for angles, stored as a dictionary.
dihedral_types: list, dict
A list of the forcefield types for dihedrals, stored as a dictionary.
