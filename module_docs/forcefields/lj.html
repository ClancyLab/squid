class squid.forcefields.lj.LJ(index=None, sigma=None, epsilon=None, line=None)[source]¶

Initialize the LJ object. Either pass index + sigma + epsilon, or pass line. If all are passed, then an error will be thrown.

This object contains the following:

  • assign_line()
  • ff_energy()
  • fix()
  • generate()
  • load_opls()
  • load_smrff()
  • pack()
  • parse_line()
  • pair_coeff_dump()
  • print_lower()
  • print_upper()
  • unpack()
  • validate()

Parameters

index: str or int
The index of the atom type.
sigma: float
Sigma in LJ expression.
epsilon: float
Epsilon in LJ expression.
line: str
A line from a parameter file to be parsed.

Returns

lj: squid.forcefields.lj.LJ
A LJ object.

validate(warn=True)[source]¶

This function will validate data integrity. In this case, we simply ensure data types are appropriate.

Parameters

warn: bool, optional
At times, weird parameters may exist (ex. sigma=0). If this is the case, we will push them to a more realistic lowerbound and print a warning (True) or crash (False).
