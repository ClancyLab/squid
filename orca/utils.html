utils¶

squid.orca.utils.get_orca_obj(parallel=True)[source]¶

This function will find the orca executable and the corresponding openmpi executable. It will handle errors accordingly.

Parameters

parallel: bool, optional
Whether we guarantee the relevant parallel openmpi is setup (True) or not (False).

Returns

orca_path: str
The path to the orca executable.
